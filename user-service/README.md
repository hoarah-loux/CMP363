# user-service

This directory contains the user microservice for the semester project.

Contents:

- `notebooks/Untitled0.ipynb` — original notebook copied verbatim (contains model, schemas, and example code).
- `app/` — FastAPI application scaffold (minimal endpoints).
- `requirements.txt` — pinned dependencies (install these inside your venv).

Quickstart:

1. Create and activate a virtual environment (you said you'll do this).
2. pip install -r requirements.txt
3. Run the app: `uvicorn app.main:app --reload --port 8000`
